# Linux File Management

This document contains basic Linux file and directory management commands with simple explanations.

---

##  File and Directory Management

- `ls` → Lists files and directories in the current location.
- `cd /path/to/directory` → Changes the working directory.
- `pwd` → Shows the current working directory path.
- `mkdir folder_name` → Creates a new directory.
- `rmdir empty_folder` → Removes an empty directory.
- `rm file.txt` → Deletes a file.
- `rm -r folder` → Deletes a folder with all its contents.
- `cp file1 file2` → Copies a file.
- `cp -r dir1 dir2` → Copies a directory recursively.
- `mv old_name new_name` → Renames or moves a file or directory.

---

## 👀 File Viewing Commands

- `cat file.txt` → Displays full file content.
- `tac file.txt` → Displays file content in reverse order.
- `less file.txt` → Opens file with scroll support.
- `more file.txt` → Displays file page by page (forward only).
- `head -n 10 file.txt` → Shows first 10 lines of a file.
- `tail -n 10 file.txt` → Shows last 10 lines of a file.

---

##  File Editing Commands

- `nano file.txt` → Opens a simple text editor in terminal.
- `vi file.txt` → Opens a powerful advanced text editor.

---

##  File Writing Commands

- `echo "Hello" > file.txt` → Writes text (overwrites existing content).
- `echo "Hello" >> file.txt` → Appends text without deleting old content.
